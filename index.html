Here are some personal projects I made.

  • Wack a Mole | Github Repo

    This was the first project I built in my bootcamp, It wasn't on the approved list of games but I was confident that I could make it work. It was made using Javascript CSS and HTML. I wanted to get practice with local storage, sounds, animations, set interval functions and DOM manipulation while using some of ES6 fun stuff. General Assembly added it to the approved list of games after I presented the finished game it felt great to make a small contribution. One student from the next class picked WhackAMole and I was able to help them started with ideas on how to build it. Overall I was really impressed with the structure of this project, All the projects had to be built these without guides or tutorials. It really is amazing what people can build when they have a place to focus on work. The game is still being worked on to be scaled for mobile.

  • Post skate Events | My Gone Go Sk8 app (or sk9) was a node.js fullstack app that was built using Express, MongoDB, Javascript and uses OAuth login with Google, I learned that it's actually really hard to get people to use apps, no one wants to sign up unless it's to post pictures of your cats. It's rare to get any users for an application. At this point in my bootcamp I realized why a team is so important to make products that people will use. I built this knowing I could tell my friends make posts, so I kept it intuitive and structured so I can use it as a reference. I hope to build more features in the future.
